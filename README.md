## Tech Stack

* Cube.js - 📊 Cube.js — Open-Source Analytical API Platform
* ve-charts - 📈 ECharts 4.x for Vue.js 2.x.
* vue-smart-widget - 🗃️ Smart widget is a flexible and extensible content container component for Vue2.x.
